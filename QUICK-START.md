# ⚡ Quick Start

## See It Now (Local Preview)
```bash
cd ~/.openclaw/workspace/brandon-health-tracker
npm run dev
# Visit http://localhost:3000
```

## Deploy It (2 minutes)
```bash
# 1. Create repo at: https://github.com/new
#    Name: health-tracker
#    Don't initialize with README

# 2. Push code
cd ~/.openclaw/workspace/brandon-health-tracker
git remote add origin https://github.com/YOUR_USERNAME/health-tracker.git
git push -u origin main

# 3. Deploy
# Visit: https://vercel.com/new
# Import your repo
# Click "Deploy"
# Done!
```

## Use It Daily

**Track water:**
- Click water drops (each = 250ml)
- Goal: 16 drops = 4L

**Track protein:**
- Click quick-add buttons (+20 eggs, +42 chicken, etc)
- Goal: 190g

**Track supplements:**
- Click checkboxes as you take them
- Goal: 8/8 daily

**Data saves automatically** in your browser.

## Add to iPhone Home Screen
1. Safari → Your Vercel URL
2. Share → "Add to Home Screen"
3. Opens like native app
